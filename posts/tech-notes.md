# 技术笔记：如何让微信分享链接变好看

## 问题

直接分享一个网页链接到微信，预览卡片往往是空白的或者很丑。这是因为微信会抓取网页的 Open Graph 标签来生成预览。

## 解决方案

在 `<head>` 中添加 meta 标签：

```html
<meta property="og:title" content="文章标题">
<meta property="og:description" content="文章描述">
<meta property="og:image" content="封面图URL">
<meta property="og:url" content="页面URL">
```

## 关键点

### 1. 封面图尺寸

微信推荐的封面图尺寸是 **900x383** 像素，比例大约是 2.35:1。

### 2. 图片必须可访问

`og:image` 的 URL 必须是完整的绝对路径，微信服务器会去抓取这张图片。

### 3. 域名备案

如果你用国内服务器，域名需要备案才能正常访问。用海外服务器（如 GitHub Pages）则不需要。

## 效果

配置好之后，分享到微信时会显示：

```
┌─────────────────────────┐
│   [封面图]              │
│                         │
│   文章标题              │
│   文章描述...           │
│   你的域名.com          │
└─────────────────────────┘
```

## 更多技巧

- 可以用 **微信 JS-SDK** 进一步自定义分享行为
- 也可以添加 `twitter:card` 标签，兼容其他平台
- 音乐播放需要用户交互后才能自动开始（浏览器策略）
