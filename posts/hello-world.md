# Hello World

> 这是我的第一篇博客文章。

## 为什么要做这个博客

我一直想拥有一个属于自己的空间，用来记录想法、技术笔记和生活感悟。不是社交媒体上碎片化的状态，而是一篇篇完整的、有深度的内容。

这个博客用最简单的方式搭建：

- **Markdown 写作** — 专注于内容本身
- **微信友好的链接** — 分享时有漂亮的卡片预览
- **自动播放音乐** — 打开就有一种氛围感
- **暗色主题** — 对眼睛友好，适合深夜阅读

## 技术栈

```
前端：原生 HTML + CSS + JS
托管：GitHub Pages（免费）
域名：可选自定义域名
音乐：HTML5 Audio API
Markdown：marked.js 解析
```

## 写作流程

1. 在 `posts/` 目录下新建 `.md` 文件
2. 用 Markdown 语法写内容
3. 在 `index.html` 的 `POSTS` 数组中添加文章信息
4. 推送到 GitHub，自动部署

就是这么简单。
